I was watching the Ramayana one evening, and a simple question came to mind:

"How tall was Kumbhakarna?"

I opened Google expecting a quick answer. Instead, I found only Quora threads, Reddit debates, and unverified blog posts. No single source cited the Valmiki Ramayana directly. No platform brought that scriptural accuracy together in one place.

That gap bothered me.

So I built one.

---

Introducing All About Ramayana — a specialized AI research platform that provides accurate, context-aware insights into the Valmiki Ramayana, Tulsidas's Ramcharitmanas, and the full tradition of the sacred epic.

The idea was straightforward: if you have a genuine question about the Ramayana — a character, an event, a philosophical teaching, a specific shloka — you should be able to get a reliable, sourced answer instantly, in whatever language you're most comfortable in.

Here is what I built:

Script-Aware Responses
The AI detects whether you type in English, Hindi (Devanagari), or Hinglish (Roman script) and responds in the exact same script. No one should have to code-switch to get an answer about their own culture.

Strict Topical Integrity
The platform is bounded purely to the Ramayana. It will not answer questions about technology, current events, or anything outside the epic. This is intentional. Precision requires focus.

Secure Backend Architecture
All AI calls route through a Firebase Cloud Function proxy. The Gemini API key lives in Google Cloud Secret Manager, never the client. Firebase App Check with reCAPTCHA v3 ensures only the website itself can call the backend.

Self-Improving Knowledge Base
Anonymized user queries are logged to Firestore and analyzed every 10 days. A training pipeline identifies knowledge gaps, synthesizes corrections, and rebuilds the contextual knowledge base. The platform improves with usage.

Model Redundancy
The system runs on a prioritized fallback chain across Gemini model versions, so even during quota pressure, the guide stays online.

---

Tech stack: React 18 + Vite | Firebase Hosting, Cloud Functions (Node.js 22), Firestore | Google Gemini API | Firebase App Check | Google Cloud Secret Manager
